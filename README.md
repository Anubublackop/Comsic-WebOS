# Neo-OS

Hi! This is Neo-OS, a simple web operating system that I made for the Hack Club Stardance challenge. 

Instead of doing a normal dark mode site or a retro look from the guide, I made a "Neo-Brutalist" style. It has bright colors, thick black lines, flat shadows, and uses code fonts to look like an indie website.

## Things I Built Into It

* **Doodle Pad:** A drawing window where you can click and move your mouse around to draw pictures on a white canvas.
* **Jiggle Mode:** A funny button inside the Start Menu. If you click it, all the open windows on your screen shake!
* **Command Hub:** A text box in the middle of the desktop. If you type words like `open calc` or `doodle`, it opens up that specific app window.
* **Calculator:** A working calculator that does simple math when you click the buttons.
* **Draggable Windows:** You can click the top of the app windows and drag them around anywhere on the screen.

## How I Coded It

* **The Design (HTML & CSS):** I wrote all of the HTML boxes and the CSS styling myself from scratch. I used simple pixel dimensions, basic borders, and solid shadows to make it look blocky.
* **The Logic (JavaScript):** I set up the basic page layout, but since I am a beginner, I used an AI tool to help me write the tricky parts of the JavaScript code. The AI helped me figure out the mouse coordinates for the window dragging logic and how to clear the HTML canvas for the drawing board.
